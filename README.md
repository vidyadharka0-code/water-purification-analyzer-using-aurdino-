
# Water Purification Analyzer using Arduino

## Description
This project analyzes water quality using multiple sensors and alerts if the water is unsafe for consumption.

## Sensors Used
- pH Sensor
- TDS Sensor
- Turbidity Sensor
- Temperature Sensor

## Components
- Arduino Nano
- LCD Display
- Buzzer
- Breadboard

## Features
- Real-time water quality monitoring
- Visual output using LCD
- Alert system using buzzer

## Purpose
This project was developed to understand sensor integration, Arduino programming, and real-world problem solving.
